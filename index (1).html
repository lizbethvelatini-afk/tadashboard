📋 How to Update Monthly

  1. Export fresh files from Greenhouse / HRIS in the same formats as before
  2. Use the upload zones below — data refreshes instantly, no page reload needed
  3. To make updates permanent on GitHub: replace data.js in your repo. Go to your repo → click data.js → pencil icon → paste the new content → commit. The data.js file is small enough to paste directly.
  4. Share the live link https://lizbethvelatini-afk.github.io/tadashboard/ — view-only for leadership
