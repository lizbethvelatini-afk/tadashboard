📋 How to Update the Dashboard

  1. Export the relevant report from Greenhouse or HRIS in the same format as before
  2. Use the upload zones below to replace the existing data — each zone corresponds to one data source
  3. Data updates instantly in the dashboard — no page reload needed
  4. To share with leadership (view-only), host this file on GitHub Pages: push index.html to a repo → Settings → Pages → Deploy. Anyone with the link can view but not edit.
  5. To update data in GitHub: replace the embedded data in the <script> section at the bottom of index.html and push the new file
