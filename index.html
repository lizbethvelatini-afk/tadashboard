📋 How to Update Monthly

  1. Export fresh files from Greenhouse / HRIS
  2. Use the upload zones below — data refreshes instantly in your browser
  3. To save permanently to GitHub: go to your repo → click index.html → pencil (Edit) icon → select all & delete → paste the new file → Commit. The file is small enough to paste now.
  4. Share https://lizbethvelatini-afk.github.io/tadashboard/ with leadership — view only
